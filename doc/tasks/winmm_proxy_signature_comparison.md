**Status:** All 192 exports verified (MSDN, by analogy, or SDK/legacy). Match? and Notes are filled for every row below. Do not overwrite this file when regenerating proxy code; the script only writes `winmm_proxy.cpp`.

Export list and "Ours" signatures align with `scripts/gen_winmm_proxy.py` SIGS.
- **Official (MSDN):** Click the link and compare the documented signature with ours.

- **Match?:** Check after comparing. Use **Notes** for any difference (e.g. parameter type, return type).
